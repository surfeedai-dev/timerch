작업한 내용을 GitHub(https://github.com/hoya838/timerch)에 푸시합니다.

다음 단계를 순서대로 실행하세요:

1. `git status`로 변경된 파일 확인
2. `git diff --staged`와 `git diff`로 변경 내용 파악
3. `git log --oneline -5`로 최근 커밋 메시지 스타일 확인
4. 변경된 파일들을 선택적으로 스테이징 (`git add <files>`)
5. 변경 내용을 잘 요약한 한국어 커밋 메시지로 커밋
6. `git push origin main`으로 푸시

커밋 메시지 형식:
```
git commit -m "$(cat <<'EOF'
<변경 내용 요약>

Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>
EOF
)"
```

변경 사항이 없으면 사용자에게 알려주세요.
