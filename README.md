# 타임캐릭터 🐾

내가 일하는 동안 화면 위에 떠 있는 데스크탑 펫.  
시간이 쌓일수록 오늘 번 돈이 실시간으로 올라가고, 유튜브나 SNS를 너무 오래 보면 캐릭터가 한마디 합니다.

> macOS 전용

---

## 설치

터미널을 열고 아래 명령어를 붙여넣으세요.

```bash
curl -sSL https://raw.githubusercontent.com/surfeedai-dev/timerch/main/install.sh | bash
```

설치가 완료되면 바탕화면에 **타임캐릭터.command** 파일이 생깁니다.  
이걸 더블클릭하면 실행됩니다.

---

## 최초 실행 시 권한 허용 (1회만)

브라우저 감시 기능을 쓰려면 macOS 손쉬운 사용 권한이 필요합니다.

1. **시스템 설정** 열기
2. **개인 정보 보호 및 보안** → **손쉬운 사용**
3. 목록에서 **터미널** 또는 **Python** 찾아서 토글 켜기

> 이 권한 없이도 앱은 실행되지만, 유튜브/SNS 감지 알림은 작동하지 않습니다.

---

## 기능

**캐릭터 오버레이**
- 화면 위에 항상 표시 (전체화면 상태에서도 유지)
- 마우스로 원하는 위치에 드래그

**시간 & 수익 추적**
- 앱을 켠 순간부터 경과 시간 카운트
- 2026년 최저시급(10,320원) 기준으로 실시간 수익 계산
- 시급은 설정에서 변경 가능

**브라우저 감시**
- 유튜브, 인스타그램, X, 틱톡, 페이스북 등 설정한 사이트를 일정 시간 이상 보면 캐릭터가 말풍선으로 알림
- 사이트 목록과 알림 시간은 설정에서 추가/삭제 가능

**설정창**
- 캐릭터 이미지를 원하는 사진으로 교체
- 시급 변경
- 감시 사이트 추가 / 삭제

> 캐릭터 위에서 **우클릭** → 설정 열기
